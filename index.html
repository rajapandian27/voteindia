National Online Voting Platform

An online voting platform is a secure, accessible digital solution that enables users to participate in elections and polls remotely, ensuring convenience, transparency, and efficiency in the voting process.


User - Self Registration

User self-registration is a process that allows individuals to create their own accounts on a platform by providing personal information, streamlining access and enhancing user autonomy without needing administrative assistance.


Required Docs:

  • National Identity Card
  • Voter ID
  • Registered Mobile Number

How to Vote ?

Follow the Steps listed below by the Authorities.


  • Login using voter credential
  • Select the Candidate
  • Poll using the OTP Verification

Application

User self-registration through our mobile application empowers individuals to easily create their own accounts anytime, providing quick access to personalized features and services with just a few taps.
